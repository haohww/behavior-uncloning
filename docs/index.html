Behavior Uncloning

Distilling Mode Redirection into Policy Weights without Inference-Time Steering

Code Citation

Abstract

Behavior-cloned robot policies often learn multiple behavior modes from diverse demonstrations, including modes that are unsafe or undesired at deployment. Behavior Uncloning studies how to edit an already-trained mixed-mode policy so it completes the task through an acceptable behavior mode. MoRE trains a temporary mode classifier, uses it to provide a differentiable mode-redirection signal during a short editing stage, preserves desired-mode competence with a retain loss, and then discards the classifier. The edited policy runs through the original inference path without inference-time steering.
